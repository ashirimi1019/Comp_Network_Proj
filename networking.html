Website Accessibility

This website is accessible online via Netlify. Domain names and IP addresses help direct users to this site.

Domain Name System (DNS)

DNS resolves domain names (e.g., example.com) into IP addresses (e.g., 192.168.1.1) so users can access websites easily.

IP Address

An IP address uniquely identifies the server hosting this website. IPv4 and IPv6 address schemes are used for communication.
